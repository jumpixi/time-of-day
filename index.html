Time of Day

A JavaScript Project by Isaac Uili

Greeting goes here

This page changes based off the time of day by utilizing JavaScript!

We are doing this by setting these times:

  • Morning (5am to 12 noon)
  • Afternoon (12 noon to 6pm)
  • Evening (6pm to 11pm)
  • Night (11pm to 5am)

      • Then we change the color, image, message to be unique for each of the 4 time periods of the day.

      The JavaScript written is connected through the HTML ID attribute within our HTML code.

      By using document.getElementById we can connect this with our JavaScript code.

      The JavaScript code gets the time from the browser-user and uses it to assign the pages current look.
